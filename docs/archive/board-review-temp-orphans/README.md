# Board-review-temp orphans (pre-archive-layout)

These 14 files were committed to `.board-review-temp/` before that directory was added to `.gitignore` and before the `docs/External-Board-Reviews/<MMDDYY>-<topic>/CONSOLIDATION.md` archive layout was adopted.

They are R1/R2 brief + agent-response artifacts from sessions 1–2 board reviews on:

- Pipeline methodology (eventually consolidated into `docs/board-review-pipeline-methodology.md`)
- Form schema review (early V1.0 / V1.1 form design)
- Progress check / data-lineage analysis

They predate the FrontierBoard SOP that says board scratch lives at `.board-review-temp/<topic>/` and gets archived to `docs/External-Board-Reviews/<MMDDYY>-<topic>/` on sign-off. Kept here for historical reference; not load-bearing for any current decision.

If you're researching the V1.0/V1.1 board-review history, the canonical archive is `docs/External-Board-Reviews/`. These orphans pre-date that layout.
